# Product Overview

Oddsparks Calculator is a web-based tool for the game **Oddsparks**. It helps players plan production chains by calculating:

- Which buildings and recipes are needed to produce a given item
- Production rates (items per minute) based on the assigned Spark builder type
- The number of buildings required to meet a target output

Players select desired output items and a builder type (Stumpy, Crafty, or Handy Spark), and the app resolves the full recipe tree showing buildings, inputs, and throughput.

The app is deployed to GitHub Pages at `/oddsparks-calculator/`.
